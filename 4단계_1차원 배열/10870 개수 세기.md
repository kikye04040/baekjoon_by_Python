# 개수 세기 ([10870](https://www.acmicpc.net/problem/10807))

> **이 문제의 핵심은 입력 받은 숫자들을 list 형태로 저장하는 법**

---

## 📘 문제 설명

n개의 정수가 주어졌을 때, 헤당 n개의 정수에 포함된 v의 개수가 몇개인지 반환하는 문제

첫줄에 입력할 정수의 개수 n을 입력받고,
둘째줄에 n개 만큼의 정수를 공백을 두고 입력.
셋째줄에 찾으려 하는 정수 v를 입력받음.

그리고 n개의 정수에 포함된 v의 개수를 출력

---

## 💡 입출력 예시

| 입력 | 출력 |
|------|------|
| `11` <br> `1 4 1 2 4 2 4 2 3 4 4` <br> `2` | `3` |

---

## 🧠 문제 풀이 아이디어

### 평소에 사용하던 map(int, input().split()) 앞에 list() 함수를 더해서 리스트로 문자열을 받기

---

# 💻 풀이 코드

```python
n = int(input())

nums = list(map(int, input().split()))

v = int(input())

count = 0

for i in n:
    if i == v:
        count += 1

print(count)
