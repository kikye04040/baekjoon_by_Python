# 공 바꾸기 ([10813](https://www.acmicpc.net/problem/10813))

> **이 문제의 핵심은 1부터 시작하는 리스트를 만들고, 1부터 출력하는 법을 아는 것 + 파이썬스러운 교환법을 아는 것**

---

## 📘 문제 설명

도현이는 바구니를 총 N개 가지고 있고,
각각의 바구니에는 1번부터 N번까지 번호가 매겨져 있다.

바구니에는 공이 1개씩 들어있고, 
처음에는 바구니에 적혀있는 번호와 같은 번호가 적힌 공이 들어있다.

도현이는 앞으로 M번 공을 바꾸려고 한다. 

도현이는 공을 바꿀 바구니 2개를 선택하고, 두 바구니에 들어있는 공을 서로 교환한다.

공을 어떻게 바꿀지가 주어졌을 때, 
M번 공을 바꾼 이후에 각 바구니에 어떤 공이 들어있는지 구하는 프로그램을 작성하시오.

---

## 💡 입출력 예시

| 입력 | 출력 |
|------|------|
| `5 4` <br> `1 2` <br> `3 4` <br> `1 4` <br> `2 2` | `3 1 4 2 5` |

---

## 🧠 문제 풀이 아이디어

### 1부터 시작하는 리스트를 만들기 위해 [i for i in range(n + 1)] 컴프리헨션 사용

### 1부터 출력하려면 range(1, n + 1)부턴 baskets[i] 출력하기

### 파이썬스러운 변수 교환 --> baskets[i], baskets[j] = baskets[j], baskets[i]

---

# 💻 풀이 코드

```python
n, m = map(int, input().split())
baskets = [i for i in range(n + 1)]

for _ in range(m):
    i, j = map(int, input().split())
    baskets[i], baskets[j] = baskets[j], baskets[i]
    
for i in range(1, n + 1):
    print(baskets[i], end=' ')
